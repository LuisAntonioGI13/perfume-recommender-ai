# Perfume Recommender AI

Sistema de recomendación de perfumes basado en similitud olfativa y búsqueda semántica con NLP.

## Features

- Recomendación por perfume base
- Recomendación por descripción natural:
  - "vainilla dulce para noche"
  - "oud elegante"
  - "cítrico fresco para calor"

## Tecnologías

- Python
- Pandas
- Scikit-Learn
- TF-IDF / Cosine Similarity
- Selenium
- Streamlit (próximamente)

## Estructura del proyecto
